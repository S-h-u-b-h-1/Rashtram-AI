# UI Improvements Report

Date: 30 June 2026

## Navigation and hierarchy

- Workspace navigation now follows: Dashboard, Bills, State Bills, Acts,
  Gazette, Policies, Profile.
- State Bills and Demo Mode are available in global search.
- The generic catalogue remains available through dashboard search without
  competing with first-class research sections.

## Catalogue experience

- Added a shared sort and direction control to the universal explorer.
- Added configurable filter names so State and Legislature/authority are clear.
- Added Open and Research actions while retaining official PDF and source links.
- Added category to result metadata and a transparent source-coverage note.
- Preserved pagination, semantic search, saved search, and multi-document chat.

## Public pages

- Replaced the broken contact form with a validated API-backed request flow.
- Replaced projected outcome claims on the Solutions page with supported
  research workflows.
- Removed decorative confidence scores and fake passage counts from the landing
  illustration.
- Reworded Product claims to describe source-aware refresh and metadata-backed
  trends.

## Consistency and accessibility

- Bills, State Bills, Acts, Gazette, and Policies use shared document and chat
  components.
- Form fields have accessible labels and status feedback.
- Desktop and 390 px mobile checks showed no horizontal overflow.
- Browser checks found no frontend console errors or Next.js error overlay.
